1. はじめに
日本のシングルマザーの貧困率は依然として高く、
その背景には「個人の努力不足」では説明できない、
構造的な誘発要因 が存在する。
その一つが、企業の転勤文化と行政制度の不整合である。

2. シングルマザーが直面する構造的リスク
シングルマザーは、以下のような“制度的に回避不能なリスク”を抱えている。
- 子育てと仕事の両立が前提
- 収入源が一つしかない
- 住居の移動が困難
- 保育・学校・地域コミュニティの安定が不可欠
- 転職のたびに生活基盤が揺らぐ
ここに 企業の転勤命令 や
行政の「自己都合離職」扱い が重なると、
生活が一気に破綻する。
つまり、シングルマザーの貧困は
“個人の問題”ではなく
制度が誘発している構造的問題 だと言える。

3. 転勤文化がシングルマザーに与える影響
転勤文化は、シングルマザーにとって特に致命的。
- 転勤に応じられない → 昇進・評価が止まる
- 拒否すると「扱いにくい人材」扱い
- 退職せざるを得ない場合、行政は「自己都合」扱い
- 失業保険の給付が遅れ、生活が不安定化
- 子どもの学校・保育園の継続が困難
- 住居の確保が難しくなる
これは、
「家庭を守るために転勤を断ると、制度が罰を与える」
という構造になっている。

4. 行政制度の問題：自己都合扱いの不整合
配偶者の転勤で辞めても自己都合。
家庭の事情で辞めても自己都合。
シングルマザーが子育てのために辞めても自己都合。
つまり行政は、
「家庭の事情はあなたの責任」

という昭和モデルを前提にしている。
しかし現代のシングルマザーは、
家庭の事情を“個人の責任”として処理できる状況にない。

5. 政策案：転勤制度と行政制度の再設計
シングルマザーの貧困を減らすには、
以下のような“構造的な予防保全”が必要。
● 5-1. 転勤の事前登録制
- 就業場所を本人が登録
- 変更には本人の同意が必要
- シングルマザーは「転勤不可」を選択可能
● 5-2. 行政の離職理由の見直し
- 子育て・家庭事情による離職を「特定理由離職」に分類
- 失業保険の給付を企業都合と同等に
- シングルマザーの生活基盤を守る
● 5-3. 企業の評価制度の見直し
- 転勤拒否を評価に影響させない
- シングルマザーのキャリア断絶を防ぐ
● 5-4. 地域定着型雇用の促進
- 地域で働き続けられる雇用モデル
- 保育・学校・地域コミュニティとの連携

6. 結論
シングルマザーの貧困は、
「努力不足」ではなく
制度が誘発している構造的な問題 である。
転勤文化と行政制度の不整合を解消し、
家庭の安定を前提とした労働環境を整えることが、
シングルマザー支援の最も効果的な政策となる。
