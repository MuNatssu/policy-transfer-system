# 背景：転勤文化と行政制度の歴史的前提

## 1. 日本の転勤文化の成立
日本の転勤制度は、高度成長期〜バブル期の以下の前提に基づいて成立した。

- 専業主婦モデル
- 単身赴任が可能な家庭構造
- 住宅価格が安く、持ち家取得が容易
- 終身雇用と年功序列
- 会社が生活を全面的に支えるモデル

これらの前提条件のもとでは、転勤は「内挿としての最適解」だった。

## 2. 現代の前提条件との乖離
現代は以下のように前提が大きく変化している。

- 共働きが標準
- 住宅価格の高騰
- 保育・学校の継続が重要
- 単身赴任は現実的でない
- キャリアは会社外にも広がる

にもかかわらず、制度は旧来の前提を外挿し続けている。

## 3. 行政制度の問題
離職理由の分類が昭和モデルのまま固定されており、

- 配偶者の転勤 → 自己都合
- 子育てのための離職 → 自己都合
- 家庭事情 → 自己責任扱い

となっている。

この構造が、家庭責任を抱える労働者を制度的に不利にしている。
